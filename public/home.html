The RM Helper App is an application that interfaces with GitHub and Jira in order to facilitate various administrative activities for Release Managers.

GitHub does not provide a means to bulk download or update key information utilized by RM's. This includes bulk downloads of pull-requests , updating labels, milestones, and releases. RM Helper fills this gap.

While Jira itself provides the ability to search, filter, and export Issues, the RM Helper App also provides a means to scrape Jira issue iD's from Github PR titles and match them agains Jira tickets in a release.

The data provided by the RM Helper App and its ability to help the administration of Github data helps release managers aggregate information for analysis, reporting, and metrics.
